# Что нового

Список изменений по сравнению с последним коммитом (`21-06-2026`).

## Карты связей (новая функция)

- Интерактивная карта на **Leaflet.js** с тайлами **CartoDB Spotify Dark**
- Цветовая дифференциация по категориям мощности:
  - ALL — синий `#3b82f6`
  - QRP (≤5W) — зелёный `#22c55e`
  - QRPp (≤1W) — оранжевый `#f97316`
  - QRP-X (≤100mW) — красный `#ef4444`
  - QRPu (≤10mW) — фиолетовый `#a855f7`
- **Радио-кнопки** для переключения категорий (одна категория за раз)
- **Линии связи** от оператора до собеседника
- **Автомасштаб** — `fitBounds` с отступом 20px, все точки в кадре
- **Масштабирование** при переключении категорий
- Жёлтый маркер позиции оператора
- Попапы с информацией: позывный, диапазон, мощность, расстояние
- **Canvas-рендерер** (`preferCanvas: true`) для производительности
- Флаг Leaflet скрыт из attribution

## Дедупликация для больших файлов

- Файлы **<10000 QSO**: каждая QSO отдельный маркер с полной информацией
- Файлы **≥10000 QSO**: объединение QSO по координатам локатора, попап показывает количество QSO, размер маркера пропорционален (log scale, 7–16px)
- Линии рисуются при ≤2000 уникальных точках

## Исправления

- **Координаты локатора** — исправлена формула `locator_to_latlon()`: убрана ошибка +1° для 6-символьных локаторов (LO48VI теперь 58.35°N, 49.79°E вместо 59.35°, 50.79°)
- **Парсинг мощности** — текстовые значения `QRP` → 5W, `QRPp` → 1W, `QRP-X` → 0.1W, `QRPu` → 0.01W
- **Форматирование мощности** — для значений <0.1W показывается 3 знака (0.008W), ≥0.1W — 1 знак (5.0W)
- **Префиксы позывных** — LA/KK6IK ищет QTH по префиксу LA (Норвегия), а не по базовому KK6IK (США)

## Прогресс анализа

- Интервал обновления: 1 секунда

## UI/UX

- **Двухколоночный макет**: слева — форма ввода, справа — результаты анализа
- Результаты растягиваются на всю высоту правого блока
- **Резиновая страница** — на всю ширину экрана
- Адаптивный дизайн: на экранах <768px блоки становятся друг под другом
- **Кнопка «Анализ»** неактивна пока не загружен файл
- **Кнопка «Скачать TXT»** неактивна и серая до завершения анализа
- Заголовок «Результат анализа» серый до анализа, зелёный после
- **Чекбокс «Не показывать карту связей»** — пропуск сбора данных карты
- Спиннер перенесён внутрь левого блока
- Радио-кнопки на отдельной строке под заголовком «Карта связей»
- Кнопка «Скачать TXT» переведена на английский
