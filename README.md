# ADIF Analyzer

Веб-приложение для анализа ADIF-логов радиолюбителей.

## Что делает

Загружаете ADIF-файл → указываете позывной и локатор → получаете статистику:

- Количество связей по категориям мощности (QRP, QRPp, QRP-X, QRPu)
- Сколько стран, полей и квадратов отработано
- Самая дальняя связь (ODX)
- Топ-20 частых позывных
- Статистика по диапазонам с ODX

## Как сделано

- **Backend**: Python + Flask + библиотеки `adif_io` (парсинг ADIF) и `pyhamtools` (определение стран/локаторов)
- **Frontend**: HTML/CSS/JS, отправляет файл на сервер через `fetch`, результат отображается на странице
- **Деплой**: GitHub → Render.com (бесплатный хостинг), автоматический деплой при пуше
