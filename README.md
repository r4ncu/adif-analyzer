# [[ADIF Analyzer]]

Веб-приложение для анализа ADIF-логов радиолюбителей и вывода статистики по странам, локаторам и мощности.

**Ссылка:** [ADIF Analyzer](https://adif-analyzer.onrender.com)

## Назначение

Программа предназначена для анализа ADIF-файлов (стандартный формат логов радиосвязи) и вывода детализированной статистики. Результаты ориентированы на таблицы активности [Клуба 72](http://club-72.ru/#qrp) — сообщества радиолюбителей, работающих малой мощностью (QRP).

## Что нужно для работы

1. **ADIF-файл (.adi)** — лог радиосвязей, экспорт из ANYLOG, UR5EQF_log, N1MM, Log4OM, MacLogger DX или другой программы ведения лога.
2. **Позывной** — ваш позывной знак (например, [[R4NCU]]).
3. **Локатор** — ваш Maidenhead-локатор 4–6 символов (например, LO48VI). Указывается вручную.

## Что можно указать

| Параметр | Обязательность | Описание |
|----------|---------------|----------|
| Позывной | Обязательно | Ваш позывной знак |
| Локатор | Обязательно | Ваш Maidenhead-локатор (4 или 6 символов) |
| ADIF-файл(ы) | Обязательно | Один или несколько файлов формата .adi |
| Мощность для всех связей | Опционально | Чекбокс + поле ввода. Если активировано, заданная мощность применяется ко **всем** QSO в логе вместо значения из поля TX_PWR файла |

## Что показывает результат

### Общая статистика по всем связям
Количество отработанных стран (WKD countries), полей (WKD fields) и квадратов (WKD squares) по **всем** QSO независимо от мощности.

### Статистика по категориям мощности

| Категория | Максимальная мощность |
|-----------|----------------------|
| QRP | до 5 Вт |
| QRPp | до 1 Вт |
| QRP-X | до 0.1 Вт (100 мВт) |
| QRPu | до 0.01 Вт (10 мВт) |

Для каждой категории:
- Количество связей
- Страны, поля и квадраты (WKD)
- Самая дальняя связь (ODX) с расстоянием и мощностью

### Дополнительно
- Топ-20 самых частых позывных
- Статистика по диапазонам (160M–2M) с указанием ODX
- Общее число уникальных позывных
- Среднее количество QSO на позывной

## Язык

Интерфейс и результаты доступны на **русском** и **английском** языках. Переключение кнопками-флагами 🇷🇺 / 🇬🇧. Выбор сохраняется между сессиями.

## Технические детали

- **Backend:** Python + Flask + `adif_io` (парсинг ADIF) + `pyhamtools` (определение стран/локаторов по базе cqdx)
- **Frontend:** HTML/CSS/JS, асинхронная обработка с поллингом
- **Деплой:** GitHub → Render.com, автоматический деплой при пуше в `main`
- **Ограничения:** Анализ 44 000 QSO занимает ~30 секунд. Фоновая обработка с опросом статуса каждые 2 секунды.
