# ADIF Analyzer — QRP-Stat

Веб-приложение для анализа ADIF-файлов (Amateur Data Interchange Format) с визуализацией связей на карте.

https://adif-analyzer.onrender.com

Результаты анализа ориентированы на таблицы активности [Клуба 72](http://club-72.ru/#qrp).

## Возможности

### Анализ ADIF-файлов

- Статистика по странам (WKD countries), полям (WKD fields) и квадратам (WKD squares)
- Категории мощности:
  - **ALL** — все связи
  - **QRP** — до 5 Вт
  - **QRPp** — до 1 Вт
  - **QRP-X** — до 0.1 Вт (100 мВт)
  - **QRPu** — до 0.01 Вт (10 мВт)
- Текстовые значения мощности: `QRP` → 5W, `QRPp` → 1W, `QRP-X` → 0.1W, `QRPu` → 0.1W
- Статистика по диапазонам (160M–2M, 70CM) с ODX (самая дальняя связь)
- ТОП-20 самых частых позывных
- Определение локатора и страны через QRZ.com (pyhamtools)
- Расчёт расстояний через Maidenhead-локаторы (pyhamtools + fallback-формула Хаверсина)

### Карта связей

- Цветовая дифференциация по категориям мощности:
  - ALL — синий
  - QRP — зелёный
  - QRPp — оранжевый
  - QRP-X — красный
  - QRPu — фиолетовый
- Радио-кнопки для переключения категорий
- Линии связи от оператора до собеседника
- Автомасштаб при загрузке и переключении категорий
- Жёлтый маркер позиции оператора
- Попапы: позывный, диапазон, мощность, расстояние
- Canvas-рендерер для производительности

### Оптимизация для больших файлов

- Файлы **<10000 QSO**: каждая QSO отдельный маркер, линии до 2000 точек
- Файлы **≥10000 QSO**: дедупликация по координатам локатора, попап показывает количество QSO, размер маркера пропорционален (log scale, 7–16px)

### Прогресс анализа

- Отображение «Обработано X из Y QSO» в реальном времени
- Интервал обновления: 1 секунда

### Интерфейс

- Двухколоночный макет: слева — форма, справа — результаты
- Результаты растягиваются на всю высоту правого блока
- Резиновая страница, адаптивный дизайн (<768px — одноколоночный)
- Кнопка «Анализ» неактивна пока не загружен файл
- Кнопка «Скачать TXT» неактивна до завершения анализа
- Чекбокс «Не показывать карту связей» — пропуск сбора данных карты
- Мультиязычность: интерфейс RU/EN, статистика RU/EN
